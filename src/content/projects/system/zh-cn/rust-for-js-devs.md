# Rust for JS Devs（英文）
这是一个给 JavaScript 开发者看的 Rust 教程项目。目前该项目只有英文版。该项目涵盖了 JS 与 Rust 的基本语法、数据类型、控制流、函数、模块、闭包、所有权、借用、生命周期对照等内容。

该项目目前出于不活跃开发状态，自 2024 年以来未更新。直到有新的议题 (issue) 被打开，该项目才会继续开发。
