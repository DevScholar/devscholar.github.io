# HTML 应用发展史

HTML 应用自万维网诞生以来经历了漫长的演进。本文将探讨网页开发的有趣发展历程。

## 起始阶段 (1990-1995)

最初，HTML 是为静态文档而设计的。早期的网站只是简单的文本页面，带有基本的格式。

## 动态网页时代 (1995-2000)

随着 JavaScript 和 CSS 的引入，网站变得更加交互。诸如以下技术开始流行：
- Java 小程序
- Flash 动画
- 动态 HTML (DHTML)

## 现代时代 (2000-至今)

如今，我们有了 React、Vue 和 Astro 等强大的框架，让构建复杂的网页应用变得更加简单。
